# Creador de test a partir de archivos PDF con feedback en tiempo real


## **SETUP**
**PACKAGES**:
create a python venv with `python -m venv venv`, activate it, and use 

    pip install -r requirements.txt

to install the required packages.

then, create a `.env` file with your google api key:

    GEMMA_API=your_api_key

and then run `pdf2questions.py`
